Welcome to Hypertext Preprocessor (PHP) tutorial.
PHP is stands for Hypertext Preprocessor. It is a very popular and widely-used of server-side scripting language. PHP scripts are executed on the server and the result will be sent to the web browser as plain HTML.

What can we do with PHP?
- generate pages and files dynamically.
- create, open, read, write and close files on the server.
- collect data from a web form such as user information, email, phone no, etc.
- send and receive cookies to track the visitor of your website.
- store, delete, and modify information in your database.
- restrict unauthorized access to your website.
- encrypt data for safe transmission over internet.

Benefits of using PHP:
- PHP is a programming language that is easy to learn and use.
- PHP is portability as it is able to be executed or run on various platforms and almost compatible with most of the servers used today.
- PHP can deliver faster results compared to other scripting languages.
