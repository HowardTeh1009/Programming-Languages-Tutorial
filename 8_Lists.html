Ordered List with different types of marker

  1. type="1" : The list items will be numbered with numbers (default).
  2. type="A" : Set the list item marker with the uppercase letters.
  3. type="a" : Set the list item marker with the lowercase letters.
  4. type="I" : Set the list item marker with the uppercase roman numbers.
  5. type="i" : Set the list item marker with the lowercase roman numbers.
